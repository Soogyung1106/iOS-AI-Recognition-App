## 1. 프로젝트 설명 
AI를 활용한 사물 인식 및 추측 어플리케이션 <br/>

## 2. 사용 기술 

- 언어 : Swift 
- IDE : Xcode 
- 프레임워크 : UIKit, Vision, AVFoundation
- ML 모델
  1) AgeNet.mlmodel
  2) CNNEmotions.mlmodel
  3) GenderNet.mlmodel
  4) Inceptionv3.mlmodel
     
(참고로 ML 모델은 용량이 너무 커서 깃허브에 업로드가 안 되어 현재 리파지토리에 업로드되어 있지는 않다.)
